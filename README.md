## Requirements
#### g++
#### make
#### SFML

## Install dependencies (Fedora)
```bash
sudo dnf install g++ make SFML
```

## Build & Run
```bash
git clone https://github.com/patrickwhelan99/CircleThingy.git
cd CircleThingy
make && ./build/circleThingy
```
